Docker Installation Guide

Follow the instructions for your specific OS to get Neo4j running locally.

Windows

Method A: Graphical Installer

Download and run the official Docker Desktop installer:

Download Docker Desktop for Windows

Method B: via Command Line (winget)

Open PowerShell as Administrator and run:

winget install -e --id Docker.DockerDesktop

macOS

Method A: Graphical Installer

Choose the version matching your chip (Intel or Apple Silicon/M1/M2):

Download Docker Desktop for Mac

Method B: via Command Line (Homebrew)

brew install --cask docker

Linux (Ubuntu/Debian)

Method A: Desktop GUI

Install Docker Desktop for Linux

Method B: Quick Script

Run the official convenience script in your terminal:

curl -fsSL https://get.docker.com -o get-docker.sh && sudo sh get-docker.sh

Final Step: Run Neo4j

Once Docker is installed and running, go back to the main lab page and run the Neo4j container command. To verify Docker is working, type this in your terminal:

docker --version
